SIMPLE C HTTP SERVER

-------

SERVER HAS BEEN TESTED AND WORKS ON

Firefox: GET, POST

Telnet: GET, POST, HEAD

Chromium: GET

--------

Telnet POST cannot have a body due to string parsing. Firefox can though, so use the form on index.html to test that.
